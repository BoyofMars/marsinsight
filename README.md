# MarsinSight

MarsinSight is a lightweight financial analytics tool that transforms raw bank statement PDFs into clear, visual spending insights.

## MVP Features

- Upload GTBank statement (PDF)
- Debit vs Credit summary
- Grouped debit analysis
- Grouped credit analysis
- Auto-generated visual charts

## Built With

- FastAPI
- Python
- Pandas
- pdfplumber
- Matplotlib

## Vision

MarsinSight aims to simplify financial visibility for individuals first, then scale toward business-level analytics and multi-bank integrations.
